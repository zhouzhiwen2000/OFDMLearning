# OFDM通信系统演示网页需求文档

## 1. 网站概述

### 1.1 网站名称
OFDM通信系统演示平台

### 1.2 网站描述
面向学生的OFDM（正交频分复用）通信系统全流程演示网页，通过交互式仿真展示OFDM系统的完整工作过程，帮助学生理解和掌握OFDM通信技术的核心原理。

## 2. 功能需求

### 2.1 核心演示功能
- 子载波生成与分配演示
- 导频插入位置配置与显示
- 多径信道仿真（可调节信道参数）
- 信道估计过程可视化
- 信道均衡算法演示
- QAM检测与解调
- 发送端和接收端星座图对比展示

### 2.2 交互式调节功能
- 支持调节子载波数量（限制为2的指数：256、512、1024、2048等）
- CP长度选择（可选多个循环前缀长度）
- 可配置导频插入密度和位置
- 多径信道参数调节（时延、衰减、相位偏移）
- 调制方式选择（QPSK、16QAM、64QAM等）
- 添加噪声干扰功能
- 随机生成多径参数功能（可指定Delay Spread）
- 实时参数调整即时反映在演示结果中

### 2.3 信道插值方式选择
- IDFT插值：只赋值导频处的信道，其余信道置零，然后做IDFT
- 极坐标线性插值（对幅度和相位分别进行线性插值）

### 2.4 多径信道配置
- 可选择路径数量（1-8条路径）
- 每条路径独立调节时延、衰减、相位偏移参数

### 2.5 导频序列配置
- 采用ZC（Zadoff-Chu）序列作为导频序列
- 支持导频序列参数配置

### 2.6 可视化展示
- 时频域信号波形显示
- 信道频率响应曲线
- 估计的信道矩阵可视化
- 发送端和接收端星座图对比
- 系统误码率(BER)性能曲线

## 3. 目标用户
通信工程、电子信息等相关专业学生，用于辅助学习OFDM通信系统的工作原理和技术细节。

## 4. 设计风格

### 4.1 配色方案
主色调采用科技蓝(#2196F3)，辅以清新绿(#4CAF50)表示正确状态，警示橙(#FF9800)提醒参数范围，错误时使用红色(#F44336)

### 4.2 视觉细节
采用现代扁平化设计，圆角半径8px，卡片式布局带有轻微阴影效果，按钮和滑块采用圆润造型，动画过渡时间为200ms

### 4.3 整体布局
左侧参数调节面板，右侧分区块展示不同功能模块的可视化结果，顶部导航栏包含功能切换，底部显示系统状态信息
