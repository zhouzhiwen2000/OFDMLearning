## 介绍

OFDM通信系统演示平台是一个面向学生的交互式教育工具，用于演示和学习OFDM（正交频分复用）通信技术的完整工作流程。

### 主要功能

- **完整的OFDM系统仿真**：从信号生成到接收解调的全流程演示
- **多种调制方式**：支持QPSK、16QAM、64QAM调制
- **信道仿真**：支持AWGN信道和多径信道仿真
- **实时可视化**：
  - 发送端和接收端星座图对比（支持ZC序列导频的圆形星座显示）
  - 子载波分布图（数据与导频位置可视化）
  - 时域和频域信号波形
  - 信道频率响应和估计
  - 误码率性能分析
- **交互式参数调节**：
  - 子载波数量和间隔
  - 导频配置（支持2的指数间隔调整）
  - 信道参数（时延、增益、相位）
  - 信噪比控制
  - 支持参数微调按钮（+/-）

### 技术特点

- 纯前端实现，无需后端服务器
- 所有计算和仿真在浏览器端完成
- 实时参数调整，即时查看结果
- 现代化UI设计，科技蓝配色方案

# npm ≥ 10

具体安装步骤如下：

## 使用说明

### 基本操作流程

1. **配置参数**：在左侧参数面板中调整OFDM系统参数
   - 基本参数：子载波数量、调制方式、导频配置、信噪比等
   - 信道参数：选择AWGN或多径信道，配置多径参数

2. **开始仿真**：点击"开始仿真"按钮执行OFDM系统仿真

3. **查看结果**：右侧区域实时显示仿真结果
   - 系统状态：误码率、调制方式、子载波数等关键指标
   - 星座图：对比发送端和接收端的信号星座
   - 信号波形：查看时域和频域信号
   - 信道响应：分析信道特性和估计效果

### 参数说明

- **子载波数量**：OFDM系统的子载波数量，限制为2的指数（64, 128, 256, 512, 1024, 2048）
- **循环前缀长度**：CP长度，用于消除符号间干扰（0 - 子载波数量/4）
- **调制方式**：
  - QPSK：4进制相移键控
  - 16QAM：16进制正交幅度调制
  - 64QAM：64进制正交幅度调制
- **导频间隔**：导频符号的插入间隔，支持2的指数（1, 2, 4, 8, 16...），间隔为1时表示全导频
- **导频功率**：导频信号的功率（0.5-2.0）
- **导频序列**：采用ZC序列（Zadoff-Chu序列），具有恒包络特性，在星座图中显示为圆环
- **信噪比**：信号与噪声的功率比（0-30 dB）
- **信道类型**：
  - AWGN：仅添加高斯白噪声
  - 多径：模拟多径传播效应
- **信道插值方法**：
  - 直角坐标线性插值：对实部和虚部分别进行线性插值
  - 极坐标线性插值：对幅度和相位分别进行线性插值
  - DFT插值：在导频位置估计信道，其余位置置零，IDFT到时延域，阈值滤波后DFT回频域，并进行幅度补偿，可有效抑制噪声
- **DFT阈值**（仅DFT插值）：时延索引阈值，高于此索引的时延分量将被置零

### 多径信道参数

对于多径信道，提供随机生成和手动配置两种方式：

**随机生成模式**
- 点击"生成随机信道参数"按钮
- **时延扩展 (Delay Spread)**：指定多径信道的最大时延扩展（1-100采样点）
- **路径数量**：指定生成的传播路径数量（1-5条），可动态调整
- 随机生成的参数遵循指数衰减模型，更符合实际信道特性
- 生成后的参数会自动填充到下方的手动配置区域，可进一步微调

**手动配置模式**
- 可通过滑块调节路径数量（1-5条），系统会自动添加或删除路径配置
- 直接调节每条路径的参数：
  - **时延**：信号到达的时间延迟（整数，0-255采样点）
  - **增益**：路径的衰减系数（0-1）
  - **相位**：路径引入的相位偏移（0-360度）
